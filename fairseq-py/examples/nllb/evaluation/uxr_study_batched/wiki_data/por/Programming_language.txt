Uma linguagem de programação de computador é uma linguagem usada para escrever programas de computador, que envolve um computador executando algum tipo de computação ou algoritmo e possivelmente controlando dispositivos externos, como impressoras, unidades de disco, robôs e assim por diante. Por exemplo, os programas PostScript são frequentemente criados por outro programa para controlar uma impressora ou tela de computador. Mais geralmente, uma linguagem de programação pode descrever computação em alguma máquina, possivelmente abstrata. É geralmente aceito que uma especificação completa para uma linguagem de programação inclua uma descrição, possivelmente idealizada, de uma máquina ou processador para essa linguagem. Na maioria dos contextos práticos, uma linguagem de programação envolve um computador; consequentemente, as linguagens de programação são geralmente definidas e estudadas dessa maneira. As linguagens de programação diferem das linguagens naturais em que as linguagens naturais são usadas apenas para a interação entre pessoas, enquanto as linguagens de programação também permitem que os seres humanos comuniquem instruções às máquinas.
